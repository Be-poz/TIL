# 알고리즘 모아보기

### 최대공약수 GCD(Grateast Common Factor)

```java
//	재귀
public long gcd(int w, int h) {
  if (h == 0) {
    return w;
  }
  return gcd(h, w % h);
}

// 반복문
public static long gcd2(int w, int h) {
  int n;

  if (w < h) {
    int temp = w;
    w=h;
    h = temp;
  }

  while (h != 0) {
    n = w % h;
    w=h;
    h = n;
  }

  return w;
}
```
